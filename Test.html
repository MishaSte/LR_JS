Для чого найкраще використовувати Grid CSS?

  • Тільки для горизонтального вирівнювання елементів
  • Для створення двовимірних макетів
  • Тільки для дизайну мобільних додатків

Які з наступних властивостей є важливими для налаштування Grid-контейнера?(оберіть кілька варіантів)

  • display: grid;
  • grid-template-columns;
  • flex-flow

Яка основна перевага використання Grid CSS?

Створити HTML структуру та CSS, щоб відобразити три блоки в ряд, використовуючи Grid CSS.
1. Створіть "div" з класом container як грід контейнер.
2. Додайте три "div" з класом box всередину контейнера.
3. Визначте "container" як грід контейнер з трьома колонками однакової ширини та розривами 10px.
4. Кожен "box" має мати фоновий колір "red", падінг 20 пікселів та центрований текст.

В чому різниця між align-content та justify-content в CSS Grid?

  • align-content вирівнює вміст сітки вздовж осі стовпчика,
    тоді як justify-content вирівнює вміст сітки вздовж осі рядка
  • align-content змінює відстань між елементами сітки,
    а justify-content змінює розмір контейнера сітки
  • align-content вирівнює вміст сітки вздовж осі рядка,
    тоді як justify-content вирівнює вміст сітки вздовж осі стовпчика

Які з наступних методів використовуються для вирівнювання елементів всередині Grid-контейнера?

  • justify-content;
  • place-items;
  • align-content;

Що таке Grid track?

  • Простір між лініями двох сусідніх рядків і двох сусідніх стовпців
  • Розділові лінії, що становлять структуру для сітки
  • Простір між двома сусідніми лініями

В яких випадках краще використовувати Grid CSS, а не Flexbox?

  • Для створення одновимірних макетів
  • Коли потрібно створити комплексний макет із багатьма рядками та стовпцями
  • Коли дизайн вимагає використання шаблонних областей
